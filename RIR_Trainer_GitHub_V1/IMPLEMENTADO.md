# Funcionalidad implementada

- Arquitectura Vite + TypeScript + Capacitor 8.
- Código dividido en pantallas, servicios y lógica.
- Vista semanal y mesociclos dinámicos de seis o más semanas.
- D1, D2 y D3 reiniciados por semana.
- Una sola fuerza por fecha.
- Cardio y pasos adicionales.
- Calentamientos específicos del entrenador.
- Aproximaciones automáticas del primer ejercicio.
- Escalera de cargas por serie con tests.
- Coach de carga en tiempo real.
- Autoguardado idempotente.
- Cola offline y sincronización automática.
- Recuperación de sesión.
- Temporizador nativo con notificación local y haptics.
- Finalización parcial o completa.
- Histórico básico y progreso.
- API nativa protegida con token rotatorio.
- GitHub Actions para test y build.
- Despliegue demo en GitHub Pages.
